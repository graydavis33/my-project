# 5) How We Communicate — Shot List

## Filming Checklist

---

### SECTION: Main Thing

**"we keep messaging pretty quick, light, and easy"**
- [ ] Talking head — Sai direct to camera, relaxed, casual setting

**"you can talk here pretty much like you would with a friend"**
- [ ] Close-up of phone screen — casual Slack DM thread scrolling

**"keep it simple, clear, and respectful"**
- [ ] Screen recording — clean Slack message being typed and sent

---

### SECTION: General

**"adding a bit of context goes a long way"**
- [ ] Screen recording — show a vague "hey" message vs a clear message side by side

**"we're all in different time zones"**
- [ ] Screen recording — hover over a Slack profile showing timezone

**"we like to use GIFs"**
- [ ] Screen recording — GIF being searched and sent in Slack

**"orange hearts to like messages"**
- [ ] Close-up screen recording — reacting to a message with the orange heart

**"custom Trendify emoji"**
- [ ] Screen recording — emoji picker showing the Trendify emoji

---

### SECTION: Channels vs DMs

**"if it helps more than one person, send it in a channel"**
- [ ] Screen recording — typing a message in a channel vs a DM

**"DMs are perfect for quick 1:1s"**
- [ ] Close-up — DM thread open on screen

---

### SECTION: Tagging

**"tag when you actually need something from someone"**
- [ ] Screen recording — @ tagging someone in a message

**"no need to tag just in case"**
- [ ] Talking head — Sai shaking head, light/casual delivery

---

### SECTION: Async Updates

**"what's done, where things are at, what's next, any blockers"**
- [ ] Screen recording — a clean async update message being typed out

**"keeps everyone aligned without needing more meetings"**
- [ ] Talking head — Sai, quick nod, confident delivery

---

### SECTION: Escalation

**"if something feels blocked or off track, raise it early"**
- [ ] Talking head — Sai leaning slightly forward, direct tone

**"share what's going on, what you've tried, and what you need"**
- [ ] Screen recording — example escalation message being typed

**"contact a manager"**
- [ ] Screen recording — DM being opened to a manager contact

---

### SECTION: Threads

**"replying in threads keeps channels clean"**
- [ ] Screen recording — clicking "reply in thread" on a Slack message

**"tag the person so they don't miss it"**
- [ ] Screen recording — @ tagging inside a thread reply

**"quick video below"**
- [ ] Screen recording — video thumbnail visible, cursor hovering over it
