# Killed Archive

When a series, format, hook style, or pillar gets cut, it gets logged here. Agents and Claude sessions query this BEFORE suggesting new ideas — if a proposal resembles something in this file, it gets flagged.

Mirrored from Obsidian vault: `content-os/killed-archive/`

---

## Entry template

```markdown
## [YYYY-MM-DD] Name of killed thing

- **Type:** series | format | hook_style | pillar
- **Ran from:** YYYY-MM-DD
- **Ran to:** YYYY-MM-DD
- **Videos produced:** N
- **Why killed:** (performance data + subjective note)
- **Replaced by:** (what rotated in, or "nothing — slot still open")
- **Retrospective lesson:** (what to remember for future attempts)
```
