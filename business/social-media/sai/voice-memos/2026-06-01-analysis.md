## Memo 2 — "SOP for Creating Shorts" (workflow vision)

**Pipeline:** record → scripts → Sai feedback → tighten together → images/graphics → (future) B-roll.

**Standout idea — AI B-roll matcher:**
- A **labeled B-roll folder** (each clip named by what it shows).
- Feed a video's script into the AI → it suggests **which B-roll fits which lines**
  (2–3 per video) and returns a **Drive link** per match.
- B-roll = **stuff they filmed**, not AI-generated. Example: a line about "reflecting"
  → pull the "walking on the street" clip.
- Currently manual: Sai types into code "find horizontal/vertical files about Sai on
  the computer, put them in a query." Organization "okay but not perfect."

**Sequencing principle:** "First do the manual process, then automate." Same for an AI
scripting system — build it only after the manual version is proven.
