## Overview

Each batch cycle follows the same pattern: **Receive → Prep → Edit → Deliver → Review → Revise**.

**Timeline:**
- **Hour 0:** Receive batch brief + footage files
- **Hour 24:** Checkpoint — progress update
- **Hour 48:** Deliver full batch (V1 draft)
- **Hour 72:** Gray reviews + gives feedback
- **Hour 120:** Deliver revisions (V2)
- **Hour 144:** Gray approves (or asks for final tweaks)

## DAY 1 — Receive & Prep (Hours 0–8)

### ✅ Step 1: Download and verify files

**Time:** 10–15 min

### ✅ Step 5: Quick checklist before starting

## DAY 1 — Edit (Hours 8–24)

### ✅ Step 6: Edit Video 1 (first video of batch)
```
[ ] Import raw footage from raw_footage/ folder into your project
[ ] Rough cut: select the best takes, remove obvious false starts
[ ] Tighten the pacing based on reference video (pace notes)
[ ] Add captions using BRAND_SPEC.md rules:
    [ ] Lowercase (except I, names, acronyms)
    [ ] 3–5 words per caption chunk
    [ ] Position based on graphics (bottom default, top if graphic at bottom)
    [ ] Font: Montserrat SemiBold 64px white
    [ ] Drop shadow: 0 4px 12px rgba(0,0,0,0.6)
    [ ] Timing: sync to word boundaries, fade in/out 80ms
[ ] Output: Video_01_DRAFT_V1.mp4 (1080×1920, 23.976fps, H.264)
```

**Drafting time:** 45 min – 1.5 hours per video (varies based on complexity)

**Checklist before moving to Video 2:**
- [ ] Video 1 is exported and saved locally
- [ ] Filename follows convention: `Video_01_DRAFT_V1.mp4`
- [ ] Duration is 30–45 sec (within target)
- [ ] Captions are readable (test on phone if possible)

### ✅ Step 7: Edit Videos 2–N (rest of batch)
```
[ ] Repeat Step 6 for each remaining video
[ ] Keep a running list of which videos are done:
    [ ] Video_01 ✓
    [ ] Video_02 ✓
    [ ] Video_03 — in progress
    [ ] Video_04 — not started
    [ ] [etc]
```

**Batching time:** 45 min – 2 hours per video

### ✅ Step 8: Quality check before delivery
```
For EACH video:
  [ ] Play full video start-to-end (catch any glitches, sync issues, caption timing)
  [ ] Check duration: 30–45 sec? (if over, tighten; if under, check if it's intentional)
  [ ] Check resolution: 1080×1920? (no black bars, no letterboxing)
  [ ] Check captions: All readable? No overlap with graphics? Proper timing?
  [ ] Check audio: No clipping, no weird gaps, volume consistent?
  [ ] Test on phone: Open video on your phone, watch it. Does it feel right?

For the BATCH:
  [ ] All videos are consistent in look/feel (not wild variation between videos)
  [ ] Captions follow the same style across all videos
  [ ] None of the videos feel like a mistake (if one seems off, re-check the brief)
```

**Time:** 30–45 min total

## Daily Habits (Best Practices)

### During editing
```
✅ Every 1–2 hours: step away (rest your eyes)
✅ Before final export: watch the whole video start-to-end
✅ Test captions on your phone (mobile viewing perspective)
✅ Keep project files organized (name your Premiere sequences, label your clips)
✅ Save frequently (backup to local drive + cloud)
```

### Communication
```
✅ Daily status update (even if just "still editing, on track")
✅ Ask questions early (don't guess)
✅ Confirm unclear feedback before revising
✅ Celebrate wins ("Video 3 looks great" etc. — Gray appreciates knowing what worked)
```

### Quality control
```
✅ Never deliver a video you haven't watched fully
✅ Never deliver with placeholder captions or missing audio
✅ Never send blurry exports or wrong resolutions
✅ If something feels off, watch a reference video again before finalizing
```

## Troubleshooting Checklist

| Problem | Solution |
|---------|----------|
| Can't open raw footage file | Confirm format is .mp4, check video codec (should be H.264) |
| Caption text looks blurry | Increase font size or move further from complex background |
| Video duration is over 45 sec | Tighten the pacing (cut more aggressively) or ask Gray if it's intentional |
| Audio sync is off | Re-sync audio to video in your editor (should be automatic on import) |
| Captions overlapping graphics | Move caption to opposite side (top if graphic at bottom, vice versa) |
| File won't upload to Drive | Check file size (should be <500MB typical), try smaller file or different browser |
| Unclear feedback from Gray | Ask in Slack immediately. Better to clarify than re-edit wrong direction. |
| Running behind on deadline | Update Slack with new ETA. No penalty, just transparency. |
| Video looks different on phone vs. monitor | This is normal. Mobile is the true format. Test on phone before final export. |
