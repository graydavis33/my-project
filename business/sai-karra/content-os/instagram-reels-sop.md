# Instagram Reels SOP (Trial Reels)

**Production tier:** LOW (raw UGC, strip-down edit)
**Cadence:** 5-10+/day via Trial Reels
**Owner:** Sai films → Gray edits → Gray posts as Trial Reels

---

## Strategy

**Trial Reels are the IG audience-test mechanism.** Each Trial Reel is shown ONLY to non-followers — so it doesn't burn your existing audience on weak tests, AND it gives a clean signal on cold-audience performance. The friend who gained 20K followers in a month posted 10×/day on Trial Reels.

- **Different videos, not different hooks of the same body.** This is the key differentiator from how Sai + Gray have tested before.
- **Same edit as TikTok / YT Shorts.** Title + captions + dead-space cuts. That's it.
- **One Trial Reel = one bet.** Don't bundle hooks.

---

## Filming (Sai)

Same as TikTok SOP. Phone, vertical, talking-head, often mid-activity. 30-90s.

---

## Editing (Gray)

Same as TikTok SOP. Strip-down: title-top + captions-bottom + cut dead space. Same export.

---

## Posting (the IG-specific part)

- Post as **Trial Reel**, not regular Reel
- **Caption:** short, Sai-voice. One sharp line.
- **Cover frame:** clear face frame, first 1s
- **Hashtags:** 3-5 niche
- **Audio:** the original voice. No trending audio overlay.
- **Schedule spacing:** spread through the day — don't dump 5 Trial Reels in an hour. Algorithm punishes burst-posting.

---

## Promotion path

If a Trial Reel performs (>3× baseline conversion to follow):
1. Promote it to a regular Reel (the IG UI offers this — accept)
2. Cross-post the same edit to TikTok + YT Shorts if not already there
3. Note the hook + body + frame in the content-alignment thesis doc — what worked, why we think so

If a Trial Reel flops:
- Leave it. It doesn't show in feed for followers. No cleanup needed.

---

## Tracking

- Trial Reel reach (non-follower views)
- Profile visits per 1K views
- **Follows per Trial Reel** — the conversion metric
- Weekly review: which Trial Reels got promoted to regular Reels? Patterns?

---

## Don't

- Don't post the same Trial Reel body 3× with different hooks — that's the old approach. New approach is different bodies entirely.
- Don't post non-Trial regular Reels except for promoted-from-Trial winners
- Don't repurpose for LinkedIn. LinkedIn is its own thing.
- Don't add B-roll, music, or HyperFrames assets
