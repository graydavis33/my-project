# YouTube Shorts SOP

**Production tier:** LOW (raw UGC, strip-down edit)
**Cadence:** Bundled with the daily 5-10+ UGC posts. YT Shorts gets every clip.
**Owner:** Sai films → Gray edits → Gray posts

---

## Strategy

YouTube Shorts is the third leg of the daily UGC tripod (TikTok + IG Reels + YT Shorts). Same edit ships to all three. YouTube Shorts has different discovery mechanics:

- **Discovery via "Shorts shelf"** — heavy reliance on first-2-second hook
- **Subscribes from Shorts feed the main YouTube channel** — which is where the long-form lives. This is the explicit funnel: Shorts → channel → long-form.
- **Cleaner caption real estate than TikTok** — title/description has room to set context

---

## Filming + Editing

Same as TikTok / IG Reels SOPs. Phone-shot UGC, title at top 5s, captions on bottom, dead-space cuts. Strip-down only.

---

## Posting

- **Title:** Sai-voice. Plain, sharp. Sets the question or claim the video answers.
- **Description:** 1-2 lines max. Links to long-form videos when relevant ("Full breakdown in this video: [link]")
- **Tags:** YouTube tags don't matter much for Shorts — skip
- **Custom thumbnail:** YouTube Shorts now supports custom thumbs. Use a clean face frame from the first second when worth the 30s. Default to auto-thumb otherwise.
- **End screen:** None on Shorts (not supported), but **the description link** can drive to the latest long-form

---

## Funnel mechanics

YouTube Shorts is the natural top-of-funnel for YouTube long-form. Each Shorts post should ideally:
- Plant a question or curiosity gap
- Reference (via description or end-of-clip line) where the longer answer lives
- Train the algorithm that Sai's audience watches both Shorts AND long-form

This is the explicit funnel design from the recap: shorts → long-form → podcast → LinkedIn.

---

## Tracking

- Views, average view duration, swipe-away rate
- **Subscribes per 1K views** — this is the YT-specific conversion metric
- Funnel check: do Shorts viewers click through to long-form? Pull this from YT Studio analytics monthly.
- Add to the social-media-analytics weekly sheet pull

---

## Don't

- Don't add visual production. Same strip-down rule as TikTok / IG Reels.
- Don't post Shorts that don't pair with the channel's identity — every Shorts post is also a billboard for the long-form channel
- Don't skip the description link to long-form when relevant — that's free funnel volume
