# Project Folder Structure

## Root
~/Desktop/my-project/

## Subfolders
- web-apps/        → HTML, CSS, JavaScript projects
- python-scripts/  → Automation, scraping, business scripts
- mobile-apps/     → iOS, Android, cross-platform apps
- business/
  - social-media/  → Social media scraping and editing tools
  - emails/        → Email automation tools
  - leads/         → Lead generation tools

## Rules
- Always save new files in the correct subfolder
- Never dump files in the root of my-project
- Keep folder names lowercase with hyphens
